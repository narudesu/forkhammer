# forkhammer new

`forkhammer new` is the primary user-facing command.

## Usage

```bash
forkhammer new -k AT-123
```

## What it does

1. Loads config from `~/.config/forkhammer/config.toml`.
2. Reads Jira issue details for the provided key.
3. Selects the configured local project root.
4. Creates a new worktree for isolated work.
5. Starts an OpenCode session and prompts issue validation.

## Requirements

- Jira credentials/config available to the CLI.
- Local OpenCode server running on `http://localhost:8000`.
- Project roots in config matching actual filesystem paths.
