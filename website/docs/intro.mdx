# forkhammer

`forkhammer` is a CLI for validating Jira issues against a local codebase and launching an OpenCode session in a dedicated git worktree.

## Quick start

Run:

```bash
forkhammer new -k AT-123
```

This command fetches issue context, resolves the target project, creates a worktree, and opens an OpenCode session focused on the ticket.
