# Configuration

Forkhammer reads local configuration from:

```text
~/.config/forkhammer/config.toml
```

## Expected settings

- Jira connection details for issue lookup.
- Project mappings used to resolve a ticket to a local repo path.
- Any defaults used when initializing an OpenCode session.

## Path sensitivity

Project root values are path-sensitive. Keep configured paths exactly aligned with your local checkout locations.
