# Configuration

Forkhammer reads local configuration from:

```text
~/.config/forkhammer/config.toml
```

## Config structure

### Secret string exchange

```toml
[opencode]
default_provider_id = "openai"
default_model_id = "gpt-5.4-mini"

[jira]
url = "https://your-jira-instance.atlassian.net"
auth = "email@example.com:api_token"

[jira.filters.inbox]
# filter_id = "12345"

[project.api]
key = "KEY"
root = "/home/you/code/forkhammer/your-repo"

[supabase]
url = "https://<project-ref>.supabase.co"
anon_key = "<anon-key>"
table = "ultrafeed_item"

[supabase.auth]
type = "secret_string"
secret_string = "<raw-cli-secret-string>"
function_url = "https://<project-ref>.supabase.co/functions/v1/generate-worker-token"
```

### Password auth

```toml
[supabase]
url = "https://<project-ref>.supabase.co"
anon_key = "<anon-key>"
table = "ultrafeed_item"

[supabase.auth]
type = "password"
email = "worker@example.com"
password = "<password>"
```
