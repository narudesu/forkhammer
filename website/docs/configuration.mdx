# Configuration

Forkhammer reads local configuration from:

```text
~/.config/forkhammer/config.toml
```

## Config structure

```toml
[opencode]
default_provider_id = "openai"
default_model_id = "gpt-5.4-mini"

[jira]
url = "https://your-jira-instance.atlassian.net"
auth = "email@example.com:api_token"

[project.api]
key = "KEY"
root = "/home/you/code/opencode/your-repo"

[supabase]
url = "https://<project-ref>.supabase.co"
anon_key = "<anon-key>"
secret_string = "<raw-cli-secret-string>"
table = "ultrafeed_item"
```
